# VTubers

## About

This is a test API server.
It's a simple, simple server that will be an endpoint for async client testing.
Do not expect much from this.

## Using

The following should all be done from within the project directory.

### Build

```
go build
```

### Run

MacOS / Linux:
```
./vtubers
```

Windows:
```
.\vtubers.exe
```
